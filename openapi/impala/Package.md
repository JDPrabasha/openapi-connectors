## Package overview
The `ballerinax/impala` is a [Ballerina](https://ballerina.io/) connector for Impala API.
This package provides the capability to search, book or sell hotels rooms.

### Compatibility
|                    | Version           |
|--------------------|-------------------|
| Ballerina Language | Swan Lake 2201.0.3| 
| Impala API         | v1.0.0            |
